
# Smart Attendance System

A web-based animated attendance management system built using **HTML, CSS, and JavaScript**.
The project works fully on **GitHub Pages** without any backend.

## Features
- Teacher & Student Login
- Attendance saving using LocalStorage
- Role-based dashboards
- Animated modern UI
- Mobile friendly

2. Upload to GitHub

4. Open the live link
